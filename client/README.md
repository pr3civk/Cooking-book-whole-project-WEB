# Cooking Book Webapp

## Installation

```bash
npm install
```

## Environment Variables

Create `.env` file in root directory:

```bash
VITE_API_BASE_URL=http://localhost:3000
```

## Development

```bash
npm run dev
```

Application runs on `http://localhost:5173`

## Build

```bash
npm run build
```

Output in `dist/` directory.

## Preview Production Build

```bash
npm run preview
```

## Lint

```bash
npm run lint
```
